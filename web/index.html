Bitchat Web IRC

A web version of the decentralized Bitchat messaging app with IRC-style commands.

Features

  • IRC-style commands (/j, /m, /w, etc.)
  • Channel-based messaging
  • Private messaging
  • Real-time peer discovery
  • Terminal-style interface

Basic Commands

  • /j #channel - Join or create a channel
  • /m @user message - Send private message
  • /w - List online users
  • /channels - Show all channels
  • /leave - Leave current channel
  • /clear - Clear chat messages
  • /help - Show all commands

Note: This is a web simulation of the original Bluetooth mesh networking functionality.
